# Web HID Sample by React

## Requirements

- Node.js: https://nodejs.jp/

## Setup Development Environments

```
$ npm install
$ npm run dev
```

- React: https://ja.react.dev/

- Material UI(MUI): https://mui.com/
- Jotai: https://jotai.org/
